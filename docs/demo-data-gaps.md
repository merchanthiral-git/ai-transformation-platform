# HR Digital Playground — Demo Data Gaps

## Part 0 — Chesapeake Demo Dataset
- Full dataset uses 200 synthetic workforce rows (reduced from spec's 8,000 for file size). Each module should still render meaningfully. If a module needs higher volume for statistical significance, generate additional rows on load.
- Task decomposition data covers top 20 roles with 5 tasks each (spec called for 15-30 per role). Sufficient for demo but not exhaustive.
- Skill proficiencies are role-level only, not person-level. Individual proficiency will need per-person data from assessments.
